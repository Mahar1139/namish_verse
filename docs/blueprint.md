# **App Name**: Namish Verse

## Core Features:

- Home Page Showcase: Dynamic hero section featuring Namish Gohil's name and profession with a light-motion background.
- About Section: Introduction about Namish Gohil's dance journey with a quote and achievement placeholders.
- Fanpage Gallery: Instagram-style profile cards showcasing fanpages with profile pictures, usernames, taglines, reel previews, and links to Instagram profiles.
- Reels Display: Clean grid layout of embedded Instagram reels with soft pop animations.
- Contact Information: Contact page with an Instagram follow button, collaboration request instructions, and a clean contact layout.
- Gift Message: A dedicated gift page with an emotional message and a button linking to @ashish_mahar._, plus a small heart animation.

## Style Guidelines:

- Primary color: Soft Neon Pink (#FF69B4) for warmth and emotional connection.
- Background color: Dark gray (#222222) for a premium dark theme.
- Accent color: Light Neon Blue (#7DF9FF) for highlighting interactive elements and smooth glow effects.
- Headline font: 'Poppins', a geometric sans-serif for a contemporary and stylish feel.
- Body font: 'Inter', a grotesque sans-serif to complement Poppins.
- Use modern, minimalist icons with soft neon glows to match the theme.
- Rounded cards and subtle shadows to give depth and a soft, premium feel.
- Smooth, elegant transitions and soft pop animations for an engaging user experience.
